Throughout the course phase, I learned numerous methods in geovisualization. These include visualization techniques, classifications, creating animations, cartographic and thematic analysis, and statistical methods used for visualizing maps. I also gained hands-on experience with various software tools and platforms, such as QGIS and ArcGIS Pro, which allowed me to apply these methods effectively.

Additionally, I utilized web technologies like HTML, CSS, and JavaScript to create interactive and dynamic maps. I explored the principles of effective map design and user interaction, ensuring that the visualizations are not only informative but also engaging and user-friendly. This comprehensive learning experience has equipped me with the skills needed to create dynamic and interactive maps that can communicate complex data in an accessible and visually appealing manner.
